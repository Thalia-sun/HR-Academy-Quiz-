# Upskill Academy

Upskill Academy ist eine interaktive Lernplattform mit Quiz Funktionen zur Vorbereitung auf berufliche Themen wie HR, Nóminas und Contabilidad.

## Live Demo
https://kutschuli.github.io/HR-Academy-Quiz-/

## Funktionen

- Interaktive Quiz Anwendung  
- Mehrsprachig (Deutsch und Spanisch)  
- Sofortiges Feedback bei Antworten  
- Einfache und schnelle Bedienung  
- Optimiert für Desktop und Mobile  

## Ziel des Projekts

Upskill Academy hilft beim strukturierten Lernen von beruflichen Inhalten und bereitet auf praktische und theoretische Kenntnisse im Arbeitsumfeld vor.

## Geplante Erweiterungen

- Weitere Lernmodule (HR, Nóminas, Contabilidad)  
- Fortschritt Tracking  
- Erweiterte Statistiken  
- Benutzerprofile  

## Nutzung

1. Seite öffnen  
2. Quiz starten  
3. Fragen beantworten  
4. Ergebnisse direkt sehen  

## Hinweis

Dieses Projekt befindet sich in aktiver Entwicklung. Funktionen können sich ändern oder erweitert werden.
